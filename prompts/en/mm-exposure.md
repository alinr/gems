## 2. Analysis of Core Touchpoints (Exposure Channels)

Here is the detailed analysis of passive touchpoints and the messages reaching the user.

**Definition of Heuristics (Priming Goal):**

- **BH1 (Category Heuristics):** The user is taught how the category works and which terms are used.
- **BH2 (Social Proof):** The user is shown that "everyone else" (peers, influencers) is already using this.
- **BH3 (Authority Bias):** The user is shown that experts (doctors, studies, trade press) validate the topic.
- **BH4 (Scarcity Bias):** The user is incidentally shown that something is limited or highly coveted.
- **BH5 (Power of Now):** (Rare in the exposure phase) The user is shown that quick solutions exist.
- **BH6 (Power of Free):** The user is incidentally offered a free entry point (e.g., a quiz, a test).
